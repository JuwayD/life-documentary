# 2026-06-10 Phase 8: 右栏搜索快捷键

## 本次目标

继续 Phase 8 信息展示优化。在已有右栏面板搜索、命中高亮和批量展开/折叠基础上，选择最小且可独立验收的后续切片：右栏搜索快捷键。

## 实现内容

- 右栏面板搜索框 placeholder 增加 `⌘/Ctrl+K` 提示。
- 前端新增键盘监听：
  - `⌘/Ctrl+K` 聚焦面板搜索框。
  - 聚焦时选中已有查询词，方便直接替换。
- 补充 Playwright E2E 测试：
  - 从主输入框按 `Control+K` 可聚焦右栏搜索框。
  - 搜索框已有内容时再次按快捷键会选中当前查询词。
- 新增 ADR-0038。

## 设计原则

- 只改前端展示层，不新增世界状态、工具或 GM Agent 规则。
- 复用现有面板搜索逻辑，不新增第二套筛选状态。
- 快捷键只负责聚焦与选择文本，搜索行为仍由原 `input` 事件驱动。
