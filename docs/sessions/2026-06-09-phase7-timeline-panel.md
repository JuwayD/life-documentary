# 2026-06-09 Phase 7: 事件时间线面板

## 本次目标

Phase 6 roadmap 已完成后,进入玩法打磨。选择最小可验收切片:在 Web 右栏展示近期事件时间线,让玩家快速回忆最近发生的世界变化。

## 实现内容

- Web 右侧状态栏新增“事件时间线”面板。
- 前端复用 `/api/state` 快照中的 `events`,展示最近 5 条事件。
- 每条事件显示事件 id/回合、可选时间戳、actor 与 summary/type/action_type。
- 补充 Web API 测试,确认快照保留最近 10 条事件且含事件 id。
- 补充 Playwright E2E 测试,确认玩家行动后时间线面板刷新。
- 新增 ADR-0019。

## 设计原则

- 不新增剧情判断:代码只展示世界事件事实。
- 不引入新后端 API:沿用已有状态快照,保持切片最小。
- 审计日志仍用于开发复盘;事件时间线面向玩家即时理解。
