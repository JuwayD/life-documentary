# 2026-06-11 Phase 9 调试快照加载确认

## 本次目标

继续打磨 Phase 9 开发者控制台调试体验,实现一个可独立验收的最小切片:测试快照加载确认摘要。

## 完成内容

- 开发者控制台测试快照“加载”改为两步确认
- 点击“加载”后在快照行内展示确认摘要
- 确认摘要包含快照名称、玩家、地点、实体/事件/Flag 数量与最近事件
- 新增“确认加载 / 取消”操作,只有确认后才恢复世界状态
- 复用已有快照详情与加载 API,不改变快照格式
- 补充 Playwright E2E 覆盖
- 新增 ADR-0076

## 验收方式

- 打开开发者控制台
- 保存一个测试快照
- 改变当前世界状态
- 点击快照“加载”
- 验证先出现“加载确认”摘要,当前世界尚未恢复
- 点击“确认加载”后,世界恢复到快照状态
