# 2026-06-10 Phase 9: 前端出生设置面板

## 本次目标

继续 Phase 9 测试与调试工具。在已有出生模板、应用工具与 reset 模板参数基础上，实现最小可独立验收的前端出生设置面板。

## 实现内容

- 顶部工具栏新增“出生设置”按钮。
- 新增出生设置弹层：
  - 左侧列出出生模板。
  - 右侧展示所选模板详情：身份、出身、HP、钱、攻防、观察、技能、初始物品与起点。
  - “以此出生开始”调用 `POST /api/reset?template_id=<id>`。
- 应用模板后：
  - 清空当前聊天。
  - 显示系统提示。
  - 按新身份/地点显示开场提示。
  - 刷新右栏状态面板与 Pixi 视口。
- 补充 Playwright E2E 覆盖模板列表、详情切换与应用后玩家/地点状态变化。

## 设计原则

- 前端只展示模板数据并调用既有 API，不判断哪个模板更适合当前剧情。
- 使用 reset 模板参数作为“新游戏选择出生”的最小闭环，避免新增并行的新游戏流程。
- 暂不做自定义模板编辑器和管理快速切换，保留为后续独立切片。
