# 2026-06-11 Phase 9 调试快照筛选

## 本次目标

继续打磨 Phase 9 开发者控制台调试体验,实现一个可独立验收的最小切片:测试场景快照筛选。

## 完成内容

- 开发者控制台测试场景快照区新增筛选输入框
- 支持按快照名称、备注、id、创建/更新时间筛选
- 增加匹配数量摘要与无匹配空状态
- 增加清空筛选入口
- 补充 Playwright E2E 覆盖
- 新增 ADR-0074

## 验收方式

- 打开开发者控制台
- 创建两个不同名称/备注的测试快照
- 在“筛选快照名称 / 备注 / id”输入关键词
- 验证列表只显示匹配快照,计数同步更新
- 清空筛选后恢复全部快照
