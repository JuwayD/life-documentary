# 2026-06-10 Phase 8: 复盘回合类型筛选

## 本次目标

继续 Phase 8 信息展示优化。在右栏快捷键提示之后，选择最小且可独立验收的后续切片：完整复盘弹层按回合类型筛选。

## 实现内容

- 复盘弹层新增回合类型筛选按钮：全部 / 剧情 / 社交 / 战斗 / 交易 / 其他。
- 每个复盘回合标题旁新增类型标签。
- 类型筛选与既有文本筛选叠加生效，并同步更新匹配数量与空结果提示。
- 归类逻辑仅基于玩家输入和 GM 叙述文本启发式判断，不新增后端接口。
- 补充 Playwright E2E 测试覆盖类型计数、标签显示、类型筛选与空结果。
- 新增 ADR-0050。
- 同步 README、docs/04-roadmap.md 与 .plan.md。

## 设计原则

- 只做复盘展示层增强，不改变世界状态、工具调用或 GM 决策。
- 与现有时间线类型筛选形成一致的信息降噪体验。
- 保持最小垂直切片，可单独验收。
