# 2026-06-10 Phase 8: 复盘筛选空状态清空入口

## 本次目标

继续 Phase 8 信息展示优化。在复盘回合类型筛选之后，选择最小且可独立验收的后续切片：优化复盘筛选无匹配时的恢复路径。

## 实现内容

- 完整复盘弹层在“无匹配回合”空状态中显示“清空筛选”按钮。
- 点击后同时清空文本筛选，并将类型筛选恢复为“全部”。
- 保留现有匹配数量、空状态与回合显示逻辑，只增加恢复入口。
- 补充 Playwright E2E 测试覆盖文本筛选空结果清空、类型+文本叠加空结果清空。
- 新增 ADR-0051。
- 同步 README、docs/04-roadmap.md 与 .plan.md。

## 设计原则

- 只做复盘展示层增强，不改变世界状态、工具调用或 GM 决策。
- 选择空结果恢复作为最小垂直切片，单独可验收。
- 与右栏上下文提示条的“清除搜索”体验保持一致。
