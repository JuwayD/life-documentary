## 关键发现

1. **小米 mimo-v2.5-pro 是真正的 Tool Use 一等公民** — 在一次回合中主动调用 11 个工具,行为高度合理。

2. **YAML 法条 + 关键词检索足够好** — Phase 0 不需要向量数据库。AI 准确区分了"在本管衙门告状(合法)"和"越诉(违法)"。

3. **审计日志价值巨大** — 每次开发完都能复盘 AI 的工具调用链,定位 prompt 优化点。

4. **完全无硬编码规则** — 18 个工具中没有一个判断"这个动作是否合法",但 AI 自己引用了 3 条法律完成判定。

## 端到端验证片段(玩家行礼呈状)

工具调用链:
```
get_entity(player) → get_entity(zhifu_wang) → list_entities_nearby(player, 5)
→ query_laws(["告状","民告官","越诉"])  ★ 法律检索
→ get_location(court_hall) → roll_dice("1d20+1")  ★ 魅力检定
→ log_event(...) → add_memory × 4  ★ 4 个 NPC 都记下了
```

引用法条: `大明律.刑律.诉讼.越诉`、`大明律.刑律.斗殴.殴制使本管长官`
叙述输出: 完整明代官场风格,知府敷衍口吻,师爷暗示,衙役肃立 — 三个建议动作收尾。
