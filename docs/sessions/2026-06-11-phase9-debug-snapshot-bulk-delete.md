# 2026-06-11 Phase 9 调试快照批量清理

## 本次目标

继续打磨 Phase 9 开发者控制台调试体验，实现最小切片: 调试快照批量清理。

## 完成内容

- 新增 `POST /api/debug/test-snapshots/bulk-delete`
- 支持按 `snapshot_ids` 去重后批量删除测试快照
- 接口返回已删除快照元数据与删除后的列表摘要
- 开发者控制台测试快照列表增加选择框、选择当前列表、已选数量、清空选择和删除所选
- 单个删除会同步清理已选状态
- 补充 FastAPI 单元测试与 Playwright E2E 覆盖
- 新增 ADR-0081

## 验收方式

- 打开开发者控制台
- 创建多个测试快照
- 使用筛选定位一组快照
- 点击“选择当前列表”后点击“删除所选”
- 验证被选快照从列表移除，未选快照保留，已选数量归零
- 运行相关测试通过
