# 2026-06-11 Phase 8 信息缺口面板

## 本次目标

继续优化 Phase 8 右栏信息展示，做一个最小可验收切片：帮助玩家快速看到当前还缺哪些信息。

## 完成内容

- 右栏新增“信息缺口”面板
- 汇总尚无线索的剧情线程
- 汇总当前地点可能遗漏的高难度观察点
- 汇总附近尚未请教的顾问
- 面板接入局势导航分组、搜索、折叠更新提示和信息密度切换
- 新增 ADR-0094
- 补充 Playwright E2E 覆盖初始缺口与线索更新后的缺口变化

## 验收方式

- 打开 Web UI，确认右栏“信息缺口”展示未触发剧情、可能遗漏细节与附近顾问
- 输入“打听线索”，确认主线已有线索后不再作为未触发剧情显示
- 运行相关测试:
  - `.venv/bin/pytest tests/web/test_e2e_browser.py -k "gaps_panel or recent_panel or summary_panel"`
