# 2026-06-10 Phase 8: 折叠面板更新提示

## 本次目标

继续 Phase 8 信息展示优化。在已有右栏折叠、搜索、批量控制和快捷键基础上，选择最小且可独立验收的后续切片：折叠面板更新提示。

## 实现内容

- 前端新增面板内容签名缓存：
  - 记录每个 `data-panel-section` 的正文文本。
  - 每次 `renderSidePanel` 后检测内容变化。
- 折叠面板内容变化时，在标题按钮右侧显示“更新”徽标。
- 用户展开该面板时自动清除更新提示。
- 新增 CSS 样式，使更新徽标使用金色提示并保持明代纸质 UI 风格。
- 补充 Playwright E2E 测试：
  - 折叠线索面板。
  - 输入触发线索记录。
  - 验证折叠标题出现“更新”。
  - 展开后验证线索可见且提示清除。
- 新增 ADR-0039。

## 设计原则

- 只改前端展示层，不新增世界状态、工具或 GM Agent 规则。
- 不主动展开面板，避免打断玩家当前阅读节奏。
- 只对折叠状态显示提示，已展开面板直接展示新内容即可。
