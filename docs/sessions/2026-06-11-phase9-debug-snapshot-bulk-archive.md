# 2026-06-11 Phase 9 调试快照批量归档

## 本次目标

继续打磨 Phase 9 开发者控制台的调试快照管理体验,实现一个非破坏性、可独立验收的最小切片。

## 完成内容

- 新增 `POST /api/debug/test-snapshots/bulk-archive`。
- 支持对所选快照批量设置 `archived=true/false`。
- 服务端返回更新后的快照元数据与列表汇总。
- 开发者控制台批量操作区新增“归档所选”和“取消归档”。
- 批量归档后清空选择并刷新列表,默认隐藏已归档快照。
- 补充服务端与 Playwright E2E 覆盖。

## 设计约束

- 仅修改调试快照元数据,不修改快照保存的世界内容。
- 不自动加载或覆盖当前世界。
- 归档用于降噪,不是删除；删除仍保留显式批量删除入口。

## 测试

- `tests/web/test_server.py` 覆盖批量归档、取消归档、空选择和未知 id。
- `tests/web/test_e2e_browser.py` 覆盖控制台勾选当前列表、批量归档、显示归档、批量取消归档。
