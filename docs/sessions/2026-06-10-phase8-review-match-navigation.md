# 2026-06-10 Phase 8: 复盘筛选匹配跳转

## 本次目标

继续 Phase 8 信息展示优化。在右栏导航当前分组高亮之后，选择最小且可独立验收的后续切片：在复盘弹层筛选结果中提供"上一个/下一个"匹配回合跳转，减少手动滚动查找的成本。

## 实现内容

- 筛选输入框旁新增匹配跳转控件（"上一个" / 索引号（如"2 / 5"）/ "下一个"）。
- 筛选条件变更时自动选中并滚动到第一个匹配回合。
- 当前匹配回合添加 `.review-match-current` 高亮样式（金色左侧框 + 浅金色背景）。
- 点击"上一个/下一个"时在匹配回合中循环跳转，索引号同步更新。
- 筛选条件恢复（无筛选或清除筛选）时隐藏跳转控件。
- 补充 Playwright E2E 测试覆盖匹配跳转功能。
- 新增 ADR-0056。
- 同步 README、docs/04-roadmap.md 与 .plan.md。

## 设计原则

- 只做前端展示层优化，不新增后端接口。
- 不改变审计日志、世界状态、工具调用或 GM 决策。
- 复用既有复盘筛选结果，不修改回合数据结构。
