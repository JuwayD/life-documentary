# 2026-06-11 Phase 9 调试快照批量导入

## 本次目标

继续打磨 Phase 9 开发者控制台调试体验，实现与“调试快照批量导出”配套的最小切片: 调试快照批量导入。

## 完成内容

- 新增 `POST /api/debug/test-snapshots/bulk-import`
- 支持导入 `mingrpg.test-snapshot-bundle` 完整快照包
- 导入前完整校验包格式、版本、快照格式与 save 结构，避免半导入
- 导入时生成新的本地快照 id，保留名称、备注和完整存档
- 开发者控制台新增“导入快照包”按钮与文件选择入口
- 导入后刷新快照列表并清空选择 / 详情 / 差异 / 加载确认状态
- 补充 FastAPI 单元测试
- 新增 ADR-0083

## 验收方式

- 创建多个测试快照
- 勾选并点击“导出所选”下载快照包
- 删除或切换当前快照列表后点击“导入快照包”
- 验证包内快照以新 id 出现在列表中
- 验证当前世界状态不会因批量导入自动改变
- 运行相关测试通过
