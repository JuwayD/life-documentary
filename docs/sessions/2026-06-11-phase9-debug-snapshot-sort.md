# 2026-06-11 Phase 9 调试快照排序切换

## 本次目标

继续打磨 Phase 9 开发者控制台调试体验,实现一个可独立验收的最小切片:测试场景快照排序切换。

## 完成内容

- 开发者控制台测试场景快照区新增排序下拉框
- 支持最近更新优先、最近创建优先、名称 A→Z、名称 Z→A
- 排序与现有快照筛选组合生效,仅影响当前前端列表展示
- 补充 Playwright E2E 覆盖
- 新增 ADR-0075

## 验收方式

- 打开开发者控制台
- 创建两个不同名称的测试快照
- 在“测试快照排序”中切换名称 A→Z / Z→A
- 验证快照列表首项按名称顺序变化
- 可继续输入筛选词,验证排序仍作用于筛选结果
