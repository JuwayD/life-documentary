# 2026-06-10 Phase 8: 信息密度切换

## 本次目标

继续 Phase 8 信息展示优化,为右栏加入“精简 / 详细”模式。玩家在推进剧情时可减少面板噪音,需要复盘时再切回详细信息。

## 实现内容

- 新增右栏信息密度切换控件:
  - 精简
  - 详细
- 使用 `localStorage` 持久化 `mingrpg.infoDensity` 偏好。
- 切换后立即重新渲染右栏,无需刷新页面。
- 精简模式调整展示:
  - 局势摘要隐藏行动角色与最新事件说明。
  - 待处理重点最多显示 2 项。
  - 关系动态最多显示 2 人,隐藏最近记忆正文,并提示剩余人数。
  - 行动建议最多显示 2 项,隐藏原因说明。
  - 事件时间线最多显示 4 条。
  - 主线剧情最多显示 3 条支线,并提示剩余支线数。
- 补充 Playwright E2E 测试:
  - 验证默认详细模式。
  - 验证切换精简后隐藏细节。
  - 验证刷新后保持精简模式。
- 新增 ADR-0033。

## 设计原则

- 只改前端展示层,不新增后端状态、接口或工具。
- 不让代码做游戏规则判断,仅改变同一快照的呈现方式。
- 保持最小安全垂直切片:HTML 控件、JS 状态与渲染、CSS 样式、E2E 验收、文档同步。
