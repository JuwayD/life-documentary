# 2026-06-10 Phase 8: 右栏面板搜索筛选

## 本次目标

继续 Phase 8 信息展示优化。根据路线中“继续优化信息展示”，选择靠前且可独立验收的最小切片：右栏状态面板搜索筛选。

## 实现内容

- 右栏信息密度切换下方新增搜索框：
  - `placeholder="筛选面板或内容"`。
  - 支持按面板标题和当前渲染内容筛选。
- 新增空结果提示：输入关键词但无匹配面板时显示“无匹配面板”。
- 新增前端逻辑：
  - `initPanelSearch` 监听输入。
  - `applyPanelSearch` 对所有 `data-panel-section` 应用显示/隐藏。
  - `renderSidePanel` 每次重渲染后重新应用当前搜索词。
- 新增样式：搜索框、空结果提示、搜索隐藏状态。
- 补充 Playwright E2E 测试：
  - 搜索“线索”时保留摘要/线索等匹配面板，隐藏玩家面板。
  - 搜索不存在内容时显示空结果。
  - 清空搜索后恢复显示。
- 新增 ADR-0035。

## 设计原则

- 只改前端展示层，不新增世界状态、工具或 GM Agent 规则。
- 搜索基于已渲染 DOM 文本，避免新增索引或后端查询复杂度。
- 保持最小安全垂直切片：HTML 入口、JS 筛选、CSS 状态、E2E 验收、文档同步。
