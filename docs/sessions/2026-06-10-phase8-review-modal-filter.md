# 2026-06-10 Phase 8: 复盘弹层筛选

## 本次目标

继续 Phase 8 信息展示优化。在已有完整复盘弹层与文本导出基础上，选择最小且可独立验收的后续切片：复盘弹层筛选。

## 实现内容

- 复盘弹层新增搜索框，支持按玩家输入、GM 叙述和回合标签筛选回合。
- 搜索框旁新增命中数量提示：
  - 默认显示当前总回合数。
  - 输入搜索词后显示匹配回合数与总回合数。
  - 无匹配时显示“无匹配回合”。
- 筛选只隐藏/显示弹层中的回合卡片，不影响文本复制、下载或审计数据。
- 补充复盘筛选样式，与现有复盘操作区保持一致。
- 补充 Playwright E2E 测试，覆盖默认总数、命中筛选、无匹配和清空搜索。
- 新增 ADR-0044。

## 设计原则

- 只改前端展示层，不新增 AI 决策逻辑。
- 复用 `/api/audit` 返回的完整回合数据，不新增后端接口。
- 筛选是定位辅助，文本导出仍保留完整复盘，避免信息丢失。
