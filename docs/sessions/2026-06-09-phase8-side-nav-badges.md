# 2026-06-09 Phase 8: 右栏导航摘要徽标

## 本次目标

继续 Phase 8 信息展示优化,在已有右栏信息导航基础上,让玩家不滚动也能看到关键分组的信息量。

## 实现内容

- 右栏信息导航按钮改为“标题 + 摘要徽标”结构。
- 新增导航徽标渲染逻辑:
  - 局势显示当前位置在场人数。
  - 行动显示当前建议入口数量。
  - 剧情显示已记录线索数。
  - 复盘显示近期事件数。
- 压力钟达到危险比例时,局势徽标高亮提示。
- 补充导航徽标样式,保持粘性导航紧凑可读。
- 补充 Playwright E2E 测试,验证初始徽标与线索/事件变化后的更新。
- 新增 ADR-0028。

## 设计原则

- 只基于现有 `/api/state` 快照聚合展示,不新增后端接口和状态字段。
- 徽标只表达“信息量”和“是否值得注意”,不替 GM Agent 做剧情或规则判断。
- 与现有行动建议面板保持一致:导航只显示建议数量,具体建议仍由原面板呈现。
