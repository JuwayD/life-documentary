# 2026-06-10 Phase 8: 右栏导航新信息汇总

## 本次目标

继续 Phase 8 信息展示优化。在已有折叠面板更新提示基础上，选择最小且可独立验收的后续切片：右栏导航新信息汇总。

## 实现内容

- 前端新增右栏导航分组与面板的映射：
  - 局势、行动、剧情、复盘分别汇总对应面板。
- 折叠面板出现“更新”提示时，对应导航入口显示 `N新` 徽标。
- 展开单个面板后自动减少导航新信息计数。
- 点击导航时优先跳转并展开该分组内第一个有更新的折叠面板，目标分组新信息徽标同步清除或减少。
- 新增 CSS 样式，使导航新信息徽标保持金色提示且不破坏现有徽标布局。
- 补充 Playwright E2E 测试：
  - 折叠线索与压力钟面板。
  - 输入触发线索与压力变化。
  - 验证“剧情”导航显示 `2新`。
  - 展开线索后降为 `1新`。
  - 点击剧情导航后清除目标更新提示。
- 新增 ADR-0040。

## 设计原则

- 只改前端展示层，不新增世界状态、工具或 GM Agent 规则。
- 不主动展开面板，保持玩家当前阅读节奏。
- 导航只做聚合提示，具体内容仍由对应面板承载。
