# 项目总纲

## 一句话定义

**一款以明朝为背景的 AI 原生角色扮演游戏:玩家通过自然语言与世界交互,AI 作为唯一的决策者,代码仅作为工具与账本。**

## 灵感来源

- **博德之门 3** — 高自由度的 CRPG 体验、回合制战斗、深度叙事
- **D&D 桌游** — AI 扮演 DM(地下城主)的角色
- **AI Dungeon** — 自然语言驱动的开放叙事
- **明代历史** — 真实的法律、礼制、社会结构作为约束与素材

## 核心愿景

打造一个**让玩家相信自己活在明朝**的游戏:
- 你可以做任何事——但每一件事都有合理的后果
- NPC 记得你做过什么,会以符合身份的方式回应
- 没有"主线/支线"的人为划分,只有你的人生轨迹
- AI 既是导演,也是世界本身

## 项目范围 (In Scope)

| 模块 | Phase 0 MVP | 完整版 |
|---|---|---|
| 输入方式 | 命令行文字 | 文字 + 鼠标点击 |
| 渲染 | 纯文字 | 2.5D 等距瓦片 |
| 世界规模 | 一个茶馆/衙门 | 一座县城及周边 |
| NPC 数量 | 3-5 个 | 30+ 个有记忆的角色 |
| 时代 | 明代万历年间 | 同上 |
| 玩家身份 | 平民书生 | 可选多种出身 |
| 战斗 | 简化回合制 | 完整回合制 |
| 存档 | JSON 快照 | 完整事件溯源 |

## 非目标 (Out of Scope)

- ❌ **不做** 多人联机
- ❌ **不做** 3D 渲染、骨骼动画
- ❌ **不做** 实时战斗
- ❌ **不做** 商业化发行(本项目是学习/分享性质)
- ❌ **不做** 历史教育产品(允许合理虚构,不追求史学严谨)
- ❌ **不做** 武侠玄幻(保持写实基调,允许民间传说级别的怪力乱神)

## 成功标准

**Phase 0 算成功,当且仅当**:
1. 玩家可以输入任何自然语言指令
2. AI 能调用工具读取世界、写入变更、生成叙述
3. 不合理的行为被 AI 合理地驳回(并给出依据)
4. 全过程有完整的可审核日志
5. 整个流程通过自动化测试

## 项目性质

- **学习项目**: 探索 AI 原生应用的开发范式
- **分享素材**: 开发过程本身是产出物
- **长期演进**: 不设交付截止日期,质量优先于速度
