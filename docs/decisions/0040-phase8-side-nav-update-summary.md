# ADR-0040: Phase 8 右栏导航新信息汇总

## 状态

Accepted

## 背景

Phase 8 已支持折叠面板更新提示。玩家折叠多个面板后，单个面板标题会显示“更新”，但如果这些面板位于滚动区域下方，玩家仍需要滚动才能知道哪个信息分组发生了变化。

## 决策

- 在右栏信息导航入口汇总对应分组内的折叠面板更新数：
  - 局势：时间、摘要、重点、玩家、场景、附近的人、关系动态。
  - 行动：行动建议。
  - 剧情：队伍、顾问、观察、主线、线索、压力钟、结局、技能。
  - 复盘：事件时间线、故事回顾。
- 当分组内存在带“更新”提示的折叠面板时，在导航按钮上显示 `N新` 徽标。
- 展开单个面板后同步减少计数；点击导航时优先跳转并展开该分组内第一个有更新的折叠面板。
- 功能仅位于前端展示层，不新增后端接口、世界状态、工具层或 GM Agent 规则。

## 后果

- 玩家无需滚动即可知道哪个信息分组出现新内容。
- 新信息提示仍保持非侵入式，不主动展开面板、不改变游戏状态。
- 分组映射是前端固定展示结构，若后续调整右栏面板，需要同步更新映射。
