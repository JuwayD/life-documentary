# ADR-0042: Phase 8 线索摘要条

## 状态

Accepted

## 背景

Phase 8 已优化右栏信息导航、搜索、折叠更新提示和时间线摘要。线索记录面板虽然已有总数、线程分组和新近标记，但玩家仍需要浏览各线程，才能快速判断“最新获得的线索是什么”和“当前最活跃的线索线程是哪一个”。

## 决策

- 在线索记录面板总览下方新增轻量摘要条：
  - 展示最新记录的线索。
  - 展示当前线索数量最多的活跃线程。
- 摘要复用已有 `story_progress`、`story_seeds` 与前端线索渲染逻辑，不新增后端接口、世界状态、工具层或 GM Agent 规则。
- 摘要只作为信息展示辅助，不改变线索排序、记录方式或 AI 决策边界。

## 后果

- 玩家能在不展开阅读全部线索的情况下快速抓住调查重点。
- 该功能保持前端展示层职责，不引入新的游戏规则判断。
- “最新线索”依赖现有线索数组记录顺序；若后续记录结构包含时间戳，可切换为时间戳排序。
