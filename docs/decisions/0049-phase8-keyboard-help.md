# ADR-0049: Phase 8 右栏快捷键提示

## 状态

Accepted

## 背景

Phase 8 已连续强化右栏的信息定位能力，包括搜索、搜索结果、快捷聚焦搜索框、折叠面板与上下文提示条。但快捷键本身只存在于输入框 placeholder 和测试中，玩家可能不知道可以用键盘快速搜索面板，也可能忽略 Enter 提交行动这一基础操作。

## 决策

- 在右栏搜索区下方新增静态“快捷键”提示块。
- 展示 `⌘/Ctrl + K` 搜索面板与 `Enter` 提交行动。
- 提示块只作为前端说明，不新增后端接口，不改变世界状态或 GM 决策。

## 后果

- 玩家能更快发现键盘操作，降低频繁定位信息时的摩擦。
- 快捷键提示与搜索框、上下文提示条保持邻近，形成右栏顶部的信息定位入口。
- 如果未来新增更多键盘操作，可在同一提示块继续补充，但应避免变成冗长说明。
