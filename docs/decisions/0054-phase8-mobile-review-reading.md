# ADR-0054: Phase 8 移动端复盘阅读优化

## 状态

Accepted

## 背景

Phase 8 已完成复盘弹层筛选、统计摘要、重点摘录与类型筛选。移动端窄屏下，复盘弹层仍沿用桌面尺寸与滚动结构，长复盘中从顶部摘要跳到最新回合需要较多滚动。

## 决策

- 在完整复盘操作区增加“跳到最新”入口。
- 点击入口时滚动到当前复盘列表的最后一个回合。
- 在窄屏下将弹层内容区域全屏化，并让复盘操作区保持粘性显示。
- 不新增后端接口，不改变审计日志、世界状态或 GM 决策逻辑。

## 后果

- 移动端查看长复盘时可快速回到最近回合，降低阅读摩擦。
- 桌面端保留原复盘结构，仅多一个可独立验收的快捷入口。
- 功能仍属于展示层优化，可与现有筛选和导出能力组合使用。
