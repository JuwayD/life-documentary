# ADR-0026: Phase 8 局势摘要面板

## 状态

Accepted

## 背景

Phase 8 继续优化信息展示质量。右栏已经支持面板折叠,但玩家仍需要在多个面板之间滚动,才能快速确认当前位置、在场人数、线索数量、压力钟和最新事件等核心局势。

## 决策

- 在 Web 右栏时间面板之后增加“局势摘要”面板。
- 摘要只汇总已有 `/api/state` 快照中的事实:当前位置、当前地点在场 NPC 数、已记录线索总数、最高压力钟、当前行动角色、最新事件摘要。
- 不新增后端 API,不改变 World、工具层、GM Agent 或审计日志行为。
- 摘要面板沿用已有折叠机制,默认展开,折叠偏好仍只保存在浏览器 `localStorage`。

## 后果

- 玩家进入页面或每次回合更新后,可先看摘要掌握局势,再按需展开细节面板。
- 摘要是展示层派生信息,不会写入存档,也不会参与游戏规则判断。
- 后续 Phase 8 可继续做面板分组置顶、筛选或更细粒度的信息高亮。
