# ADR-0025: Phase 8 右栏面板折叠

## 状态

Accepted

## 背景

Phase 8 继续优化信息展示质量。随着剧情、线索、压力钟、队伍、顾问、观察、结局、技能和复盘等面板陆续加入,Web 右栏信息密度变高。玩家在不同阶段关注的面板不同,但当前所有面板始终展开,容易造成滚动负担和信息过载。

## 决策

- 为右栏每个信息面板增加标题按钮,点击可折叠或展开该面板内容。
- 折叠状态保存到浏览器 `localStorage`,刷新页面后保留玩家偏好。
- 只影响浏览器展示层,不改变 `/api/state` 数据结构、World、工具层或 GM Agent 行为。
- 默认全部展开,保证首次访问仍能看到完整信息。

## 后果

- 玩家可按当前关注点收起暂时不需要的面板,减少右栏滚动和视觉噪音。
- 偏好仅保存在本机浏览器,不会写入游戏存档或世界状态。
- 后续可继续追加分组置顶、摘要模式或重要信息高亮等 Phase 8 展示优化。
