# ADR-0041: Phase 8 时间线摘要

## 状态

Accepted

## 背景

Phase 8 已支持事件时间线筛选。玩家可以按剧情、社交、战斗、交易、其他筛选事件，但筛选后仍需要浏览按钮和事件列表，才能判断当前视图包含多少事件、近期事件主要集中在哪一类。

## 决策

- 在事件时间线面板顶部新增轻量摘要：
  - 展示当前筛选范围下的事件数量。
  - 展示最近事件中数量最多的事件类型。
- 摘要复用前端已有的事件类型识别与筛选计数，不新增后端接口、世界状态、工具层或 GM Agent 规则。
- 摘要随筛选按钮即时更新，保持与现有时间线筛选交互一致。

## 后果

- 玩家无需逐条浏览即可快速理解近期事件结构。
- 该功能只优化信息展示，不改变游戏状态与 AI 决策边界。
- 事件类型仍由前端展示层启发式识别；若后续事件结构更明确，可替换为后端提供的类型字段。
