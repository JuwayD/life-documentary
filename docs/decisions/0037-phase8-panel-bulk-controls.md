# ADR-0037: Phase 8 面板批量展开/折叠

## 状态

Accepted

## 背景

Phase 8 已加入右栏面板折叠、信息密度切换、移动端抽屉、搜索筛选与搜索命中高亮。玩家可以逐个收起信息面板，但当右栏内容很多时，仍需要多次点击才能快速切换“只看标题”和“展开详读”两种浏览状态。

## 决策

- 在右栏信息密度切换下方新增“全部展开 / 全部折叠”批量控制。
- 批量控制复用现有 `mingrpg.panelCollapsed` 本地持久化：
  - “全部折叠”将所有 `data-panel-section` 标记为折叠。
  - “全部展开”将所有 `data-panel-section` 标记为展开。
  - 每个面板的 `aria-expanded` 与可见状态同步更新。
- 抽出前端展示层 helper `setPanelCollapsed`，供单面板切换、侧栏导航自动展开、批量控制共同复用。
- 不新增后端接口、世界状态、工具层或 GM Agent 规则。

## 后果

- 玩家可快速压缩右栏信息，先扫标题，再按需展开。
- 该切片完全停留在前端展示层，风险低，可通过 Playwright E2E 独立验收。
- 批量状态会覆盖单个面板此前的折叠偏好，符合“批量控制”的直觉语义。
