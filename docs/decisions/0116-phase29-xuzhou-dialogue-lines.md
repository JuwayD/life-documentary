# ADR-0116: Phase 29 补充 — 徐州 NPC 对话素材

## 状态

已接受

## 背景

Phase 28 引入了 NPC 对话系统（dialogue_lines），为 12 个扬州 NPC 添加了对话素材。Phase 29 新增了徐州地域和 3 个 NPC，但未包含 dialogue_lines。为保持对话系统完整性，需要为徐州 NPC 补充对话素材。

## 决策

为徐州 3 个 NPC 添加 dialogue_lines：

### 钱知府 (xuzhou_magistrate)
- **角色定位**: 犹豫的官员，收到朝廷密令但左右为难
- **对话风格**: 官方口吻，谨慎措辞，高态度时透露内幕
- **关键话题**: 朝廷密令、官商勾结
- **叙事作用**: 高态度时暗示去问周老五，形成调查线索

### 周老五 (xuzhou_innkeeper)
- **角色定位**: 消息灵通的掌柜，八面玲珑但嘴紧
- **对话风格**: 热情但谨慎，高态度时透露关键情报
- **关键话题**: 南来北往、铁器生意
- **叙事作用**: 高态度时暗示半夜去西门，引导调查方向

### 孙铁商 (xuzhou_iron_merchant)
- **角色定位**: 走私铁器的商人，与吴员外有生意往来
- **对话风格**: 警惕、表面正经，高态度时承认暗中勾当
- **关键话题**: 正经生意、暗中勾当
- **叙事作用**: 高态度时暗示去问知府，形成调查闭环

## 理由

1. **对话完整性**: 所有关键 NPC 都应有 dialogue_lines，确保对话系统一致
2. **叙事连贯性**: 三个 NPC 的对话形成调查闭环，引导玩家逐步深入
3. **角色多样性**: 官员、掌柜、商人三种角色，提供不同角度的信息
4. **独立可验**: 3 个 NPC 的 dialogue_lines 可独立测试验证

## 后续

- 可为其他地域 NPC 补充 dialogue_lines（镇江、淮安等）
- 可增加更多话题和特殊台词，丰富对话深度
- 可根据玩家调查进度动态解锁对话选项
