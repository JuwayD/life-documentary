# ADR-0046: Phase 8 右栏搜索结果列表

## 状态

Accepted

## 背景

Phase 8 已完成右栏搜索、命中高亮和命中数量。玩家输入关键词后能知道哪些面板匹配，但仍需要在过滤后的右栏中滚动确认命中位置；如果目标面板之前处于折叠状态，还需要额外展开。

## 决策

- 在右栏搜索框下方新增搜索结果列表。
- 每条结果展示匹配面板标题和一段内容片段。
- 点击结果时自动展开对应面板、滚动到该面板并触发已有高亮焦点样式。
- 结果列表只基于当前前端已渲染的面板 DOM 计算，不新增后端接口和游戏规则判断。

## 后果

- 玩家可以从搜索框直接跳到目标面板，减少滚动与展开成本。
- 搜索体验形成“数量总览 → 结果入口 → 面板内高亮”的完整定位链路。
- 改动保持在前端展示层，不影响状态结构、AI 决策或审计日志。
