# ADR-0057: Phase 8 最近变化面板

## 状态

Accepted

## 背景

Phase 8 已完成局势摘要、待处理重点、事件时间线、线索摘要、压力钟行动联动等信息展示优化。玩家在长会话或中断后回到游戏时，仍需要分别查看时间线、线索与压力钟，才能恢复“刚刚发生了什么”的上下文。

## 决策

- 在右栏“待处理重点”之后增加“最近变化”面板。
- 面板聚合展示最新事件、最新线索与当前最高压力钟。
- 每条变化以“类别 / 摘要 / 元信息”三段展示，保持轻量可扫读。
- 面板只读取当前 `/api/state` 快照，不新增后端接口、不写世界状态。
- 将该面板纳入“局势”导航分组与折叠更新提示体系。

## 后果

- 玩家可在一个面板中快速了解最近发生的事件、线索和压力变化。
- 该功能与现有局势摘要、待处理重点互补：摘要回答“现在在哪/有什么”，最近变化回答“刚发生了什么”。
- 纯前端展示增强，不影响 GM 决策、工具调用、审计日志或世界状态。
