# ADR-0093: Phase 9 调试快照健康问题定位

## 状态

Accepted

## 背景

调试快照健康校验已经能列出损坏、重名或结构缺口快照,并可导出诊断报告。但当快照列表较长、存在筛选或归档隐藏时,调试者需要手动复制问题快照 id 再查找,定位成本仍然偏高。

## 决策

在健康校验结果中为每个问题快照提供“定位”入口：

- 健康报告的问题列表为每条问题追加定位按钮。
- 点击后自动用问题快照 id 筛选快照列表。
- 定位时临时显示归档快照,避免问题快照因默认隐藏而找不到。
- 目标快照短暂高亮并滚动到视野中。

该能力只改变开发者控制台的筛选/展示状态,不修改快照内容,不加载存档,不修复或删除文件。

## 后果

- 调试者可从健康问题直接跳到对应快照,减少坏档排查的手动步骤。
- 健康校验从“发现问题”补齐到“定位问题”的最小闭环。
- 保持调试工具只读定位语义,不会产生额外数据风险。
