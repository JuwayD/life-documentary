# ADR-0033: Phase 8 信息密度切换

## 状态

Accepted

## 背景

Phase 8 持续优化右栏信息展示。当前右栏已经包含局势摘要、待处理重点、关系动态、行动建议、时间线、剧情与复盘等面板。虽然折叠与导航已降低查找成本,但玩家在游玩时仍可能只想保留关键摘要,而复盘时才需要完整细节。需要一个最小、可逆、纯展示层的密度切换。

## 决策

- 在右栏导航下方加入“精简 / 详细”信息密度切换。
- 使用 `localStorage` 持久化偏好,刷新后保持选择。
- 精简模式只影响前端展示密度:
  - 局势摘要隐藏行动角色与最新事件说明。
  - 待处理重点与行动建议限制展示数量。
  - 行动建议隐藏原因文本。
  - 关系动态隐藏最近记忆正文并限制展示人数。
  - 时间线减少最近事件数量。
  - 主线剧情限制支线展示数量。
- 不修改后端接口、世界状态、工具层或 GM Agent 规则。

## 后果

- 玩家可在“游玩时看重点”和“复盘时看细节”之间快速切换。
- 该能力完全可逆,不会影响存档、审计日志或 AI 判断。
- 后续若新增面板,可按同一密度模式补充展示规则。
