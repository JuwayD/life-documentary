# ADR-0085: Phase 9 调试快照标签管理

## 状态

Accepted

## 背景

Phase 9 已支持测试快照保存、筛选、排序、导入导出与批量操作。随着快照数量增加，仅靠名称和备注筛选不够稳定，开发者需要用轻量元数据归类复现盘面。

## 决策

为测试快照新增 `tags` 元数据字段：

- 创建快照时可提交标签列表。
- 编辑快照元数据时可更新标签，且不改变快照内的世界存档内容。
- 列表、详情、单个导出、索引导出、批量导出/导入预览均保留标签。
- 前端开发者控制台支持逗号分隔录入标签、展示标签胶囊、按标签下拉筛选。

标签只作为调试快照元数据，不进入世界状态，也不参与任何游戏规则判断。

## 后果

- 大量测试快照可按“府衙 / 街市 / 回归 / 冒烟”等维度快速归类。
- 分享快照与快照包时保留标签，便于接收方定位用途。
- 旧快照没有 `tags` 字段时按空标签处理，保持兼容。
