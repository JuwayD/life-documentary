# ADR-0043: Phase 8 右栏搜索命中数量

## 状态

Accepted

## 背景

Phase 8 已提供右栏面板搜索与命中高亮。玩家输入搜索词后能看到不相关面板被隐藏，但仍需要自行判断当前命中范围：到底只有一两个面板命中，还是多数面板都包含相关信息。

## 决策

- 在右栏搜索框下方新增轻量命中数量提示：
  - 未搜索时显示当前总面板数。
  - 搜索时显示“匹配 N / M 个面板”。
  - 无匹配时继续显示既有“无匹配面板”提示。
- 该提示复用现有 `applyPanelSearch` 对面板的遍历结果，不新增后端接口、世界状态、工具层或 GM Agent 规则。
- 提示只作为信息导航辅助，不改变搜索匹配规则、面板内容或折叠状态。

## 后果

- 玩家能快速判断搜索词覆盖范围，减少误以为信息缺失的情况。
- 功能保持前端展示层职责，风险低且可独立验收。
- 面板总数来自当前 DOM 中的 `data-panel-section`，后续新增面板会自动计入。
