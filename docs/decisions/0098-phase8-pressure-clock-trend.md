# ADR-0098: 压力钟趋势可视化

**状态**: 已接受
**日期**: 2026-06-11

## 背景

压力钟面板只显示当前值和危险线,玩家无法判断局势是在恶化还是好转。需要增加趋势可视化,让玩家一目了然。

## 决策

1. **后端历史追踪**: `advance_pressure_clock` 在现有 `value`/`danger_at` 基础上新增 `history` 数组,每次推进时追加新值。
2. **前端趋势箭头**: 比较 `history` 最后两个值,显示 ↑(上升)/↓(下降)/→(持平) 趋势指示器。
3. **前端迷你折线图**: 用 SVG `<polyline>` 绘制历史值走势,颜色与压力等级(安全/警告/危险)一致。

## 后果

- 历史数组会随游戏进程增长,但压力钟数量有限(通常 2-5 个),内存影响可忽略。
- 旧存档兼容:无 `history` 字段时静默降级为不显示趋势。
- 向前兼容:GM Agent 可通过观察 history 判断趋势,辅助叙事决策。
