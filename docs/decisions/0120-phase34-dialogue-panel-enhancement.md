# ADR-0120: Phase 34 对话面板增强

## 状态

已接受

## 背景

Phase 30 实现了对话历史面板基础功能（NPC 态度趋势、话题筛选、时间线展示）。随着游戏进展，对话记录日益增多，玩家需要更高效的搜索、统计和导出能力。

## 决策

在现有对话历史面板上增加四项增强功能：

1. **对话内容搜索** — 文本输入框实时过滤对话记录（匹配玩家台词和 NPC 回应），显示匹配计数。
2. **态度分布可视化** — 汇总所有 NPC 态度值，以水平堆叠条形图展示友善/中立/冷淡分布。
3. **话题统计** — 按对话次数排序的水平条形图，展示 Top 5 话题。
4. **导出对话历史** — 一键下载 `.txt` 文件，包含全部对话记录、态度变化和回合信息。

此外，NPC 态度卡片的 sparkline 升级为大尺寸（120x32），显示数据点圆圈和 hover tooltip。

## 技术方案

- 前端纯 JS/CSS 实现，无新增后端 API
- 搜索通过 `data-searchable` 属性存储玩家台词+NPC 回应的小写文本，input 事件触发 DOM 过滤
- 导出使用 Blob + 下载链接，客户端生成文本
- 态度分布从已有 `attitudeHistory` 对象计算，复用现有数据结构

## 后果

- 正面：玩家可快速定位对话、了解态度全局分布、导出分享
- 无新增后端依赖，纯前端变更
