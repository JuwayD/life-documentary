# ADR-0079: Phase 9 调试快照索引导出

## 状态

Accepted

## 背景

ADR-0077/0078 已支持单个测试快照导出与导入。随着快照数量增加,调试时还需要快速归档“有哪些快照、各自规模与备注”,但不必导出完整世界存档内容。

## 决策

为测试快照增加索引导出能力:

1. 新增 `GET /api/debug/test-snapshots/export-index`,返回 `mingrpg.test-snapshot-index` JSON。
2. 索引只包含快照元数据与实体/地点/事件/Flag 计数,不包含完整 `save` 内容。
3. 响应包含 `snapshot_count`、汇总 `totals`、`exported_at` 与快照摘要列表。
4. 开发者控制台测试快照区域增加“导出索引”按钮,下载索引 JSON 并显示系统提示。

## 后果

- 可以一次性归档当前调试快照目录,便于对照、分享测试覆盖清单。
- 索引文件体积小,不会泄露完整世界状态细节。
- 如需复现具体盘面,仍使用单个快照导出文件。
