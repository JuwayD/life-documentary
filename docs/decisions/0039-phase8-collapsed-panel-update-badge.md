# ADR-0039: Phase 8 折叠面板更新提示

## 状态

Accepted

## 背景

Phase 8 已支持右栏面板折叠、批量展开/折叠、搜索与快捷键。玩家压缩信息面板后，面板内容仍可能随着回合推进发生变化；如果折叠状态下没有提示，玩家需要逐个展开才能发现新线索、状态或事件。

## 决策

- 为折叠的右栏面板加入内容更新提示：
  - 前端记录每个 `data-panel-section` 的正文文本签名。
  - `renderSidePanel` 后比较本次与上次签名。
  - 只有面板处于折叠状态且内容发生变化时，才在标题按钮右侧显示“更新”徽标。
  - 面板被展开时自动清除该提示。
- 保持功能在前端展示层，不新增后端接口、世界状态、工具层或 GM Agent 规则。

## 后果

- 玩家可以长期折叠低频面板，同时仍能看到哪些区域有新信息。
- 更新提示不会打断当前行动，也不会改变任何游戏状态。
- 该机制依赖面板文本内容签名，适合当前以信息展示为主的右栏结构。
