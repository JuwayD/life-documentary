# ADR-0044: Phase 8 复盘弹层筛选

## 状态

Accepted

## 背景

Phase 7 已提供完整复盘弹层与文本导出，Phase 8 持续优化信息展示。随着回合增多，玩家在完整复盘中查找某次行动或 GM 叙述片段需要滚动浏览，复盘定位成本上升。

## 决策

- 在复盘弹层内新增轻量搜索框：
  - 未搜索时显示当前回合总数。
  - 输入关键词后按玩家输入、GM 叙述和回合标签筛选回合。
  - 搜索时显示“匹配 N / M 个回合”。
  - 无匹配时显示“无匹配回合”。
- 筛选仅作用于弹层内 DOM 展示，不改变审计日志、世界状态、导出文本或 AI 决策逻辑。
- 复用 `/api/audit` 已返回的回合数据，不新增后端接口。

## 后果

- 玩家能在长复盘中快速定位关键回合。
- 文本导出仍保持完整故事记录，避免因筛选造成误导。
- 功能保持前端展示层职责，风险低且可独立验收。
