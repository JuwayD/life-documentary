# ADR-0032: Phase 8 右栏导航目标高亮

## 状态

Accepted

## 背景

Phase 8 已增加右栏信息导航、导航徽标、待处理重点、关系动态与事件时间线筛选。导航可以快速滚动到目标面板并自动展开,但右栏内容密集时,玩家跳转后仍需要重新确认当前定位到了哪个面板。需要补一个最小展示层反馈,降低定位成本。

## 决策

- 右栏导航按钮点击目标面板后,为目标 `panel-section` 添加短暂 `focused` 状态。
- `focused` 状态使用浅金色背景和左侧金色内阴影提示当前位置。
- 高亮自动移除,不持久化、不影响折叠偏好。
- 继续只改前端展示层,不新增后端接口、状态字段或工具。

## 后果

- 玩家通过导航跳转后能立即识别目标面板,减少右栏滚动后的迷失感。
- 行为保持轻量、可逆,不会影响世界状态、审计日志或 GM Agent 判断。
- 后续若增加更多导航分组,可复用同一高亮机制。
