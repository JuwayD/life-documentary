# ADR-0122: Phase 36 宁波地域

## 背景

Phase 0-35 已完成 8 个地域（扬州、瓜洲渡、南京、苏州、杭州、镇江、淮安、徐州）共 57 个 NPC。宁波作为明代浙江东海大港，以天一阁藏书楼和海上走私贸易闻名，是"江南暗网"调查链的出海终端。

## 决策

新增宁波地域，含 3 个地点、3 个 NPC、1 条支线。

### 地点

| ID | 名称 | 类型 | 说明 |
|---|---|---|---|
| `ningbo_dock` | 宁波码头 | outdoor | 东海港口，渔船商船云集，暗中走私装卸 |
| `ningbo_shipsi` | 宁波市舶司 | indoor_official | 海上贸易管理衙门，负责查验船只、稽查走私 |
| `ningbo_tianyi` | 天一阁 | indoor_cultural | 范钦所建藏书楼，藏书七万余卷 |

### NPC

| ID | 姓名 | 角色 | 关键信息 |
|---|---|---|---|
| `ningbo_fanqin` | 范钦 | 致仕兵部右侍郎、天一阁主人 | 掌握海禁政策和江南商帮旧档案，与扬州官场有书信往来 |
| `ningbo_chief` | 陈把总 | 市舶司把总 | 暗中收茶水银放行走私，知道"福顺号"船 |
| `ningbo_sea_merchant` | 赵海商 | 海商 | 江南商帮在宁波的代理人，与杭州/苏州/扬州吴员外有往来 |

### 路由

杭州拱宸桥码头 (`hangzhou_canal_dock`) → 宁波码头 (`ningbo_dock`)，出口键 `southeast_ningbo`。

### 支线

"海上暗线" (`ningbo_sea_route`)：追查宁波出海口的走私终端，串联杭州、苏州、扬州吴员外。

### 跨地域联动

- 宁波 3 个 NPC 和 3 个地点加入"江南暗网" (`jiangnan_network`) 锚点
- 新增调查里程碑 `nb_sea_route`（海上暗线）
- 吴员外 `network_regions` 扩展至 8 个地域

## 替代方案

- 方案 A：将宁波作为杭州的子区域 → 拒绝，宁波有独立的历史和地理意义
- 方案 B：增加更多地点（如月湖、城隍庙）→ 3 个地点已足够覆盖核心功能

## 影响

- 后端：`server.py` 注册 `seed_ningbo`，`agent.py` 添加宁波 NPC 描述，`phase23_main_story.py` 扩展调查网络
- 测试：24 个新增覆盖测试
