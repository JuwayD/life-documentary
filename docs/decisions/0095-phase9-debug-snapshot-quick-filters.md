# ADR-0095: Phase 9 调试快照快速筛选摘要

## 状态

Accepted

## 背景

开发者控制台已经支持快照名称/备注/id/标签筛选,也支持置顶、归档与标签管理。但常用调试场景仍需要在多个控件之间切换:查看当前可见快照、找置顶快照、打开归档快照、按高频标签筛选。快照数量增长后,这些入口不够直观。

## 决策

在测试场景快照区增加一条“快速筛选摘要”入口:

- 展示当前可见、全部、置顶、归档数量。
- 展示数量最高的前 6 个标签及其快照数。
- 点击“当前/全部/置顶/归档”会同步现有文本筛选与归档显示状态。
- 点击标签芯片会复用既有标签筛选状态并显示归档快照,便于找全同类盘面。

该能力只改变开发者控制台展示状态,不修改快照内容,不加载或删除存档。

## 后果

- 调试者可从摘要直接进入高频筛选路径,减少重复输入。
- 快照标签、置顶、归档的已有能力获得更明显的入口。
- 继续保持前端最小增强,不新增后端 API 或数据结构。
