# ADR-0115: Phase 29 — 新地域 (徐州)

## 状态

已接受

## 背景

项目已完成 Phase 0-28,7 个地域覆盖江南运河沿线。为进一步扩展世界广度和调查深度,需要添加北方战略要地。

## 决策

添加徐州作为第 8 个地域:

### 地点 (3)
1. **徐州城门** (xuzhou_city_gate) — 五省通衢的咽喉,南来北往的商旅必经之地
2. **徐州府衙** (xuzhou_yamen) — 淮北地区最高官署,知府大人在此处理政务
3. **徐州驿站** (xuzhou_inn) — 官道上最大的客栈,消息集散地

### NPC (3)
1. **钱知府** (xuzhou_magistrate) — 徐州知府,进士出身,为官谨慎但魄力不足
2. **周老五** (xuzhou_innkeeper) — 驿站掌柜,消息灵通,对各路客商了如指掌
3. **孙铁商** (xuzhou_iron_merchant) — 铁器商人,与吴员外有生意往来

### 支线素材 (1)
- **铁器暗运** (xuzhou_iron_route) — 铁器从扬州经徐州北运,避开官府盘查

### 地域连接
- 淮安清江浦 → 徐州城门 (运河/陆路)

### 跨地域调查更新
- merchant_wu network_regions 扩展至 7 地
- 新增调查里程碑: xz_iron_route (铁器暗运)
- 江南暗网支线锚点增加徐州 NPC 和地点

## 理由

1. **地理完整性**:徐州是大运河与南北陆路交汇的战略要地,连接江南与北方
2. **叙事价值**:铁器走私为调查增添新维度,从日用品角度揭示暗网的北方运输线
3. **角色多样性**:知府的犹豫、掌柜的消息、铁商的暗线,提供丰富的互动选择
4. **独立可验**:3 地点 + 3 NPC + 1 支线,结构完整,可独立验收

## 后续
