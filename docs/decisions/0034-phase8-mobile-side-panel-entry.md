# ADR-0034: Phase 8 移动端右栏信息入口

## 状态

Accepted

## 背景

Phase 8 持续优化信息展示。桌面端右栏已经有折叠、导航、摘要、重点、关系、时间线筛选与信息密度切换；但窄屏布局中 `.side-panel` 直接隐藏，移动端玩家只能看到对话区，无法查看局势摘要、行动建议、线索和复盘入口。

## 决策

- 在顶部操作区加入“状态面板”按钮，仅在窄屏显示。
- 窄屏下将原有右栏改为抽屉式侧栏：
  - 默认从右侧隐藏。
  - 点击“状态面板”滑入显示。
  - 点击遮罩关闭。
  - 按钮维护 `aria-expanded` 状态。
- 复用现有 `.side-panel` 内容、面板折叠、信息密度、导航和渲染逻辑。
- 不新增后端接口、世界状态、工具层或 GM Agent 规则。

## 后果

- 移动端玩家可随时查看右栏信息，而不会占用主对话区空间。
- 桌面端布局保持不变。
- 后续若继续优化移动端，可在同一抽屉入口上增强关闭按钮、手势或底部导航。
