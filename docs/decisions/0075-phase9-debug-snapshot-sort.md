# ADR-0075: Phase 9 调试快照排序切换

## 状态

Accepted

## 背景

开发者控制台已经支持测试场景快照筛选。随着快照数量增加,只依赖默认时间顺序和文本筛选,在对比多个命名相近的回归盘面时仍不够直观。

## 决策

在开发者控制台的测试场景快照区增加本地排序下拉框:

- 最近更新优先
- 最近创建优先
- 名称 A→Z
- 名称 Z→A

排序只作用于已加载且已按当前关键词筛选后的快照列表,不改变后端 API、不修改快照数据、不新增规则判断。

## 后果

- 调试快照较多时可按名称或时间快速定位
- 与现有筛选功能组合使用,继续打磨 Phase 9 调试体验
- 保持纯前端展示增强,不影响存档格式和快照接口
