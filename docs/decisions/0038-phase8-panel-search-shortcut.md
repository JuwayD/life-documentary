# ADR-0038: Phase 8 右栏搜索快捷键

## 状态

Accepted

## 背景

Phase 8 已加入右栏面板搜索、搜索命中高亮与面板批量展开/折叠。玩家可以通过搜索框定位信息，但仍需要先移动鼠标点击搜索框，键盘流玩家在对话输入与状态查询之间切换不够顺手。

## 决策

- 为右栏面板搜索加入 `⌘/Ctrl+K` 快捷键：
  - 按下快捷键时聚焦 `#panel-search-input`。
  - 若搜索框已有内容，则选中当前文本，便于直接替换查询词。
- 在搜索框 placeholder 中提示快捷键。
- 保持功能在前端展示层，不新增后端接口、世界状态、工具层或 GM Agent 规则。

## 后果

- 玩家可从主输入框快速跳到状态面板搜索，减少鼠标操作。
- 选中已有搜索词使连续查询更快。
- 快捷键只拦截常见搜索组合，不影响普通输入与现有面板筛选逻辑。
