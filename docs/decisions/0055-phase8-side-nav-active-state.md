# ADR-0055: Phase 8 右栏导航当前分组高亮

## 状态

Accepted

## 背景

Phase 8 已完成右栏信息导航、目标跳转高亮、搜索、折叠更新提示与移动端右栏入口。随着右栏面板数量增加，玩家虽然能通过导航跳转到局势/行动/剧情/复盘分组，但滚动或跳转后缺少持续的当前位置提示。

## 决策

- 为右栏导航按钮增加当前分组 active 状态与 `aria-current`。
- 点击导航时立即高亮对应分组。
- 右栏滚动时根据当前可见面板所属分组同步导航高亮。
- 复用现有分组映射，不新增后端接口、不改变世界状态或 GM 决策逻辑。

## 后果

- 玩家在长右栏中更容易判断当前正在查看哪个信息分组。
- 键盘/读屏辅助可通过 `aria-current` 获得当前位置语义。
- 功能仍为纯展示层增强，可与搜索、折叠和移动端抽屉组合使用。
