# ADR-0023: Phase 7 文本复盘导出

## 状态

Accepted

## 背景

Phase 7 已有故事回顾弹层,可以按回合查看玩家输入与 GM 叙述。下一步打磨需要降低分享门槛,让玩家不用复制审计日志或截图,即可把游玩过程整理成纯文本。

## 决策

- 在故事回顾弹层顶部新增“复制文本”和“下载文本”操作。
- 文本内容复用 `/api/audit?limit=50` 返回的玩家输入与 GM 叙述,按回合顺序生成 Markdown 风格纯文本。
- 下载在浏览器端生成 `.txt` 文件,不新增后端 API。
- 不加入剧情判断、线索筛选或摘要生成;代码只重排已有事实。

## 后果

- 玩家可以把复盘内容直接复制到聊天、笔记或分享文档。
- 文本导出与审计日志仍保持分层:玩家看到故事文本,开发者继续通过审计弹层查看工具细节。
- 当前导出范围限制为最近 50 回合,与现有复盘弹层一致。
