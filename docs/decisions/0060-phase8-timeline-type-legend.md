# ADR-0060: Phase 8 时间线类型图例

## 状态

Accepted

## 背景

Phase 8 已为事件时间线加入类型筛选、摘要、按回合分组与组内类型统计。玩家可以看到“剧情/社交/战斗/交易/其他”等标签，但这些分类的含义需要从事件内容中推断，首次阅读时仍有理解成本。

## 决策

- 在事件时间线摘要与筛选按钮之间增加类型图例。
- 图例说明五类事件的用途：
  - 剧情：线索、结局与主线推进。
  - 社交：请教、交谈与人物互动。
  - 战斗：攻击、受伤与战斗后果。
  - 交易：买卖、雇佣与钱物变化。
  - 其他：移动、观察及未归类记录。
- 图例仅解释既有前端分类，不改变 `detectTimelineType` 逻辑，不写入世界状态。
- 复用现有颜色体系，以小圆点和短说明降低阅读负担。

## 后果

- 玩家更容易理解时间线筛选按钮与事件标签的含义。
- 信息展示更自解释，减少依赖外部说明或试错点击。
- 纯 UI 增强，不影响 GM Agent、工具层、审计日志或存档格式。
