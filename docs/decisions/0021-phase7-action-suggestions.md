# ADR-0021: Phase 7 行动建议入口

## 状态

Accepted

## 背景

Phase 7 的目标是降低游玩摩擦。事件、线索、顾问、观察等面板已经提供信息,但新玩家仍需要把这些信息手动转化为下一步输入。

## 决策

- 在 Web 右栏新增“行动建议”面板,放在附近人物与事件时间线之间。
- 建议入口只根据当前快照中的事实生成可点击行动模板:观察、询问在场者、请教附近顾问、梳理线索或查看出口。
- 点击建议仍通过普通 `handleTurn` 发送自然语言输入,由 GM Agent 判断与执行,前端不新增规则裁决。
- 不新增后端 API,继续复用 `/api/state`。

## 后果

- 玩家可以从当前局势直接发起常见行动,减少空白输入框带来的停顿。
- 建议是入口而非答案,不会替代 LLM 的叙事与规则判断。
- 后续可继续扩展建议文案,但必须保持“基于事实展示、不做剧情判断”的边界。
