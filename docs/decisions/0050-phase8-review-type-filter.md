# ADR-0050: Phase 8 复盘回合类型筛选

## 状态

Accepted

## 背景

Phase 8 已增强完整复盘弹层的统计摘要、重点摘录与文本筛选。随着回合增多，玩家仍需要在复盘中区分剧情、社交、战斗、交易等不同类型的回合，单纯文本搜索不足以快速缩小阅读范围。

## 决策

- 在完整复盘弹层中新增回合类型筛选按钮。
- 按玩家输入与 GM 叙述的文本特征轻量归类为剧情、社交、战斗、交易、其他。
- 在每个回合标题旁展示类型标签。
- 筛选与既有文本搜索叠加生效，不新增后端接口，不改变世界状态或 GM 决策。

## 后果

- 玩家可按回合性质快速复盘关键片段，减少长记录中的阅读噪音。
- 类型识别保持前端展示层的启发式分类，避免把分类结果写入状态或规则层。
- 后续若审计日志提供结构化类型，可替换前端启发式来源。
