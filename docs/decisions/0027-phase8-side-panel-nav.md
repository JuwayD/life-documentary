# ADR-0027: Phase 8 右栏信息导航

## 状态

Accepted

## 背景

Phase 8 已加入局势摘要与面板折叠,但右栏信息继续增多后,玩家仍需要滚动寻找常用区域。需要一个不改变规则、不新增后端的轻量导航入口,帮助玩家在信息密集面板中快速定位。

## 决策

- 在 Web 右栏顶部增加粘性“信息导航”。
- 导航提供 4 个分组入口:局势、行动、剧情、复盘。
- 点击入口时滚动到对应面板,并自动展开目标面板。
- 继续复用已有折叠状态存储,不新增 API、World 字段、工具或 GM Agent 规则。

## 后果

- 玩家在右栏任意滚动位置都能快速回到关键分组。
- 折叠后的面板可以通过导航直接恢复可见,降低“找不到信息”的摩擦。
- 导航只是展示层行为,不写入存档,不影响游戏决策。
