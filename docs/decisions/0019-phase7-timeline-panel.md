# ADR-0019: Phase 7 事件时间线面板

## 状态

Accepted

## 背景

Phase 6 roadmap 已完成。下一步进入玩法打磨,优先选择可独立验收且不引入新规则判断的小切片:让玩家在 Web 右栏看到近期世界事件,帮助理解最近发生了什么。

## 决策

- 复用 `World.snapshot()` 中已有的最近事件数据,不新增状态表或规则逻辑。
- 前端右侧新增“事件时间线”面板,展示最近 5 条事件。
- 每条事件显示事件 id/回合、可选时间戳、actor 与 summary/type/action_type。
- 事件仍由 GM Agent 或工具层写入;前端只负责展示事实。

## 后果

- 玩家可以不用打开审计日志,直接看近期世界变化。
- 该功能不影响 GM 决策,也不解释事件意义。
- 未来如需完整回放,可在此基础上增加分页或筛选,但当前保持最小切片。
