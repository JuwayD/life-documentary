# ADR-0067: Phase 8 线索线程进度概览

## 状态

Accepted

## 背景

Phase 8 已为右栏增加线索摘要条、主线剧情、线索记录与事件联动。玩家能看到具体线索，但在多条主线/支线并行时，仍需要逐段阅读才能判断线索集中在哪些线程、哪些线程尚未触发。

## 决策

- 在主线剧情面板顶部增加“线索线程进度概览”。
- 概览展示：
  - 线索总数。
  - 活跃线程数 / 全部线程数。
  - 未触发线程数。
- 当已有线索时，展示最多 4 个（精简模式 2 个）线索最多的线程及线索数量。
- 数据只来自 `story_seeds` 与 `story_progress`，不推断任务完成度，不写入世界状态。
- 保持纯前端信息展示增强，不改变 GM Agent、工具层、审计日志或存档格式。

## 后果

- 玩家能更快判断剧情线索分布与待探索方向。
- 主线/支线信息层级更清晰，减少在长线索列表中扫读的负担。
- 该增强不引入新的游戏规则判断，符合“代码只展示数据、AI 做决策”的项目原则。
