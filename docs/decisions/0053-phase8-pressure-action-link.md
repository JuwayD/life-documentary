# ADR-0053: Phase 8 压力钟行动联动

## 状态

Accepted

## 背景

Phase 8 已将压力钟展示在局势摘要、待处理重点与压力钟面板中。玩家看到压力接近危险线后，仍需要自己组织下一步输入，信息提示与行动入口之间存在一步转换成本。

## 决策

- 在压力钟达到半数危险线时，于压力钟卡片显示“应对行动”按钮。
- 按压力程度生成自然语言行动：半数以上提示“需要尽快处理”，接近危险线提示“当务之急”。
- 点击按钮直接提交一条自然语言行动，请 GM 处理该压力钟相关线索、人物安抚与局势降温。
- 不新增后端接口，不改变压力钟规则、世界状态结构、工具调用或 GM 决策逻辑。

## 后果

- 玩家能从高风险信息直接进入可执行行动，降低信息到行动的摩擦。
- 该功能仍是展示层联动，实际是否能降温由 GM Agent 与工具调用决定。
- 低压力钟不显示行动按钮，避免过度提示。
