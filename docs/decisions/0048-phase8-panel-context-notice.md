# ADR-0048: Phase 8 右栏上下文提示条

## 状态

Accepted

## 背景

Phase 8 已完成右栏搜索、搜索结果、折叠面板更新提示与批量展开/折叠。玩家可以快速隐藏不相关信息，但当搜索或折叠生效时，右栏顶部缺少一个整体提示来说明当前有多少信息被隐藏，以及如何恢复完整视图。

## 决策

- 在右栏搜索区下方新增上下文提示条。
- 当搜索隐藏部分面板时，提示隐藏数量，并提供“清除搜索”按钮。
- 当存在折叠面板时，提示折叠数量，并提供“全部展开”按钮。
- 提示条只读取当前 DOM 面板状态，不新增后端接口，不改变世界状态。

## 后果

- 玩家能更快意识到当前右栏不是完整视图，降低误判信息缺失的概率。
- 搜索与折叠的恢复入口更靠近筛选控件，减少滚动和定位成本。
- 提示条由前端状态驱动，若未来新增面板，只要使用既有 `data-panel-section` 结构即可自动纳入统计。
