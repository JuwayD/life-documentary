# ADR-0022: Phase 7 回放/复盘体验增强

## 状态

Accepted

## 背景

Phase 7 已经加入事件时间线、信息层级和行动建议。玩家在多回合游玩后仍需要一个低摩擦入口,快速回看自己做过什么、GM 如何回应,用于复盘线索和分享游玩过程。

## 决策

- 在 Web 右栏新增“故事回顾”面板,展示当前快照中的基础复盘指标:回合数、线索数、涉及 NPC 数。
- 增加“查看完整回顾”按钮,打开复盘弹层。
- 复盘弹层复用现有 `/api/audit?limit=50`,按时间顺序展示玩家输入与 GM 叙述。
- 不新增后端 API,不新增剧情规则判断;代码只重排已有审计日志与状态快照。

## 后果

- 玩家可以在不打开审计技术细节的情况下回看故事经过。
- 审计日志继续服务开发排查,故事回顾服务玩家复盘与分享。
- 当前实现是最小切片,后续可加入导出文本或按线索筛选,但仍应复用已有事实数据。
