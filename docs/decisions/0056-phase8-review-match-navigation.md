# ADR-0056: Phase 8 复盘筛选匹配跳转

## 状态

Accepted

## 背景

Phase 8 已完成复盘筛选功能（文本/类型筛选）与复盘空状态清空入口。玩家已能在复盘弹层中按文本或回合类型筛选回合，但仍需手动滚动查找匹配回合位置。当筛选结果较多时，逐一滚动查找效率低，缺少"上/下一个匹配"快速定位手段。

## 决策

- 在复盘筛选输入框旁增加匹配跳转控件（"上一个" / 位置索引 / "下一个"）。
- 当前匹配回合使用高亮样式（金色左侧框 + 浅金色背景，对齐既有 `.focused` 风格）。
- 使用 `.review-match-current` 类标记当前选中的匹配回合。
- 筛选条件变更时自动定位到第一个匹配回合。
- 复用现有筛选结果列表，不改变世界状态、工具调用或 GM 决策逻辑。

## 后果

- 玩家在筛选后可一键在上/下一个匹配回合间跳转，提高长复盘中的定位效率。
- 匹配索引显示（如"3 / 7"）让玩家清楚当前在全部匹配中的位置。
- 点击"上一个/下一个"时自动滚动画布到目标回合，减少手动滚动。
- 功能为纯前端增强，不对 audit 数据做任何修改。
