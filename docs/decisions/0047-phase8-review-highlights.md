# ADR-0047: Phase 8 复盘重点摘录

## 状态

Accepted

## 背景

Phase 8 已完成复盘弹层统计摘要、筛选和右栏搜索结果。玩家可以查看完整回合和按文本过滤，但在记录变长后，打开复盘时仍需要从完整时间线里自行判断最近几步的关键行动与结果。

## 决策

- 在完整复盘弹层的统计摘要下方新增“重点摘录”。
- 摘录最近 3 个回合，按新到旧展示回合号、玩家行动短句和 GM 叙述短句。
- 摘录仅使用 `/api/audit` 已返回的 turn 数据，不调用 LLM，不新增后端接口。
- 完整时间线、筛选、复制和下载文本保持原有行为。

## 后果

- 玩家打开复盘即可先看到最近关键行动和结果，减少滚动成本。
- 摘录是确定性的前端展示摘要，不影响世界状态、AI 决策或审计日志。
- 长文本会被截断，详细内容仍以完整回合时间线为准。
