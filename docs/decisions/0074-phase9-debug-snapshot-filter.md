# ADR-0074: Phase 9 调试快照筛选

## 状态

Accepted

## 背景

开发者控制台已经支持保存、加载、预览、编辑、删除测试场景快照。随着调试快照数量增加,仅按时间倒序浏览会降低定位效率。

## 决策

在开发者控制台的测试场景快照区增加本地筛选输入框:

- 按快照 `id`、名称、备注、创建/更新时间进行大小写不敏感匹配
- 展示“共 N 个快照”或“匹配 X / N 个快照”
- 无匹配时显示空状态
- 提供清空筛选入口

该功能只筛选已加载的快照列表,不改变快照数据、不新增后端规则判断。

## 后果

- 快照多时可更快定位回归盘面
- 保持 Phase 9 调试工具的前端体验打磨方向
- 不影响现有快照 API 与存档格式
