# ADR-0045: Phase 8 复盘统计摘要

## 状态

Accepted

## 背景

Phase 8 已完成复盘弹层筛选，玩家可以在长复盘中定位具体回合。下一步仍沿信息展示优化推进：打开完整复盘时，玩家还需要先滚动阅读才能判断本次记录规模、最近停在哪一步，以及玩家/GM 内容大致体量。

## 决策

- 在完整复盘弹层顶部新增轻量统计摘要：
  - 总回合数。
  - 玩家输入字数。
  - GM 叙述字数。
  - 最近回合编号与玩家输入摘要。
- 摘要只基于 `/api/audit` 已返回的回合数据在前端计算。
- 不改变审计日志、世界状态、文本复制/下载内容或 AI 决策逻辑。

## 后果

- 玩家打开复盘后可以先获得整体规模与最新进度，再决定搜索或滚动查看细节。
- 功能保持前端展示层职责，不新增后端接口。
- 与既有复盘筛选互补，形成“先总览、再定位”的复盘体验。
