# ADR-0051: Phase 8 复盘筛选空状态清空入口

## 状态

Accepted

## 背景

Phase 8 已为完整复盘弹层加入文本筛选与回合类型筛选。两类筛选叠加后，玩家可能进入“无匹配回合”状态；如果只展示空结果提示，玩家需要自行判断是文本条件、类型条件还是二者叠加导致的空白。

## 决策

- 在复盘弹层“无匹配回合”空状态中新增“清空筛选”按钮。
- 点击后同时清空文本筛选，并将类型筛选恢复为“全部”。
- 复用现有前端筛选逻辑，不新增后端接口，不改变审计日志、世界状态或 GM 决策。

## 后果

- 玩家从空结果状态回到完整复盘只需一次点击，减少筛选叠加带来的困惑。
- 该功能保持展示层增强性质，可独立验收。
- 后续若复盘筛选条件继续增加，可沿用同一清空入口统一恢复默认状态。
